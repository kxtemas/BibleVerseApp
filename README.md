Hi, welcome to my CST-247 Bible Search Assessment!
===================================================================================================================================

This assessment was created to be able to create project where users are able to search for a bible verse by inserting the Book Name, Chapter Number and Verse Number and even selecting a transation version such as ESV and NIV.
------------------

This repository contains the code built to create a Bible Verse Search project by using a restAPI and C#.
